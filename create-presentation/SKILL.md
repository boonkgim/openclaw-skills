# Presentation Slide Builder

The template is a single HTML file (`template.html` next to this SKILL.md) with slides containing `{{PLACEHOLDER}}` text. Your job:

1. Copy the file to `presentation/[yyyy-mm-dd]-[slug].html`
2. Replace placeholders with real content given by the user
3. Tell the user to open the HTML file in their browser.
