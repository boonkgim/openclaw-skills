# openclaw-skills

Public skill library for [OpenClaw](https://github.com/boonkgim/openclaw-installer).

## Skills

| Skill | Description |
|-------|-------------|
| [create-image-cloudflare](./create-image-cloudflare) | Generate images via Cloudflare Workers AI (Flux Schnell). Free tier — ~2,000 images/day. |
| [telegram-media](./telegram-media) | Send images or media to Telegram via CLI when direct attachment fails. |
